Pallete

About This Project.

Palletes is a single page javascript application that allows user to upload and search color schemes based on criteria such as categories and the base color. Schemes can be created, updated, and deleted by the user that created the scheme. The colors of the scheme can be copied and pasted.

What Technologies Does this Application Use?

Hosted on: Heroku

Front-End

  • React
  • Bootstrap 5
  • Axios
  • _ Lodash

Back-End

  • Node.js
  • Express
  • Passport
  • MongoDB
